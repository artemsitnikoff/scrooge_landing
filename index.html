Пример интерфейса — загрузка и статус передачи

Загрузка из Excel или JSON

Привезли данные
на флешке?

Выгрузите файл с весами прямо в Telegram-бот. SCROOGE проверит формат, валидирует каждую запись и передаст данные в ФГИС УТКО. Есть интернет — уйдёт сразу. Нет — сохранит и отправит когда появится.

  • Принимает Excel и JSON в формате ФГИС УТКО (API v1)
  • Проверяет госномер ТС, вес въезда/выезда, время, код ФККО
  • Дубли по ID выгрузки пропускаются автоматически
  • Сохраняет очередь при отсутствии интернета — отправит сам
  • Уведомление в Telegram о каждом успешном акте в ФГИС УТКО
Возможности

Всё — через Telegram

Управляйте несколькими полигонами из одного чата. Ни одной веб-страницы.

💾

Флешка или файл

Выгрузите данные с весов на носитель, привезите к интернету и загрузите в бот — SCROOGE сам передаст в ФГИС УТКО

🛰

Офлайн-очередь

Загрузили файл, но интернет пропал? Данные сохранятся в боте и уйдут в ФГИС УТКО автоматически при появлении связи

📊

Excel и JSON

Принимаем файлы в формате ФГИС УТКО API v1. Поддерживаются все обязательные поля: госномер, вес, время, код ФККО

🏭

Несколько объектов

Управляйте всеми полигонами и сортировочными комплексами в одном боте — статус по каждому

🔔

Мгновенные уведомления (Push-уведомления)

Акты отправлены, ошибка валидации, восстановлена связь — всё приходит в Telegram мгновенно

🛡

Валидация данных

Автопроверка каждой записи: госномер ТС (формат М000ММ55), вес въезда/выезда, дата в UTC, код ФККО — до отправки в ФГИС

🔑

🔒 Ваш ключ доступа — только у вас

Ключ доступа (accessKey) хранится в зашифрованном виде. Мы не имеем доступа к вашим данным в ФГИС УТКО
